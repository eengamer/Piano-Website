# Piano Portfolio - Github Pages

This is a personal piano portfolio website built with **React** and **Vite**, designed to be deployed on **Github Pages**.

## Project Type

- **Framework**: React + Vite
- **Deployment**: Github Pages (Static Frontend)
- **Base URL**: https://yourusername.github.io/Piano-Website/

## Key Features

✨ **Luxury Design** - Professional, elegant styling with gold accents
🎹 **Parallax Effects** - Smooth scrolling parallax backgrounds
✨ **Scroll Animations** - Fade-in, slide-in, and zoom effects as sections come into view
📱 **Responsive** - Mobile-friendly design for all devices
⚡ **Fast Performance** - Optimized Vite build for quick loading

## Building for Github Pages

The project is configured with `base: '/Piano-Website/'` in `vite.config.js` to work correctly when served from a subdirectory on Github Pages.

To build: `npm run build`

The `dist/` folder contains the static files ready for deployment.

## Deployment

This repository is set up to deploy automatically to Github Pages. The `dist/` folder should be pushed to the `gh-pages` branch (or configured in repository settings).

See deployment instructions in the repository README for more details.
